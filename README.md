## What I Learned

Through this assignment, I learned how to:

- Create and clone Git repositories.
- Organize files and folders in a project.
- Track changes using Git.
- Stage and commit changes with meaningful commit messages.
- Create and switch between branches.
- Merge branches into the main branch.
- Push local repositories to GitHub.
- Create and merge Pull Requests.
- Manage project versions using Git and GitHub.

## Git Commands Used During the Assignment

```bash
git clone <repository-url>

git status

git add .

git add <filename>

git commit -m "Commit message"

git log

git diff

git branch

git checkout -b feature/about-me

git checkout -b feature/projects

git checkout main

git merge feature/about-me

git push origin main

git push origin feature/about-me

git push origin feature/projects

git branch -d feature/about-me

git branch -d feature/projects

git pull origin main
```
